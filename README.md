# Liquid Programming

## A Tamplating Language

<hr>

[Link](URL)

### **Start End Condition**

```bash
{% %}
```

### **Output print**

```bash
{{ }}
```

### **Variable Declear**

```bash
{{ assign variable_name = variable_value }}

    Examples:
    --------
    {{ assign myName = "Shama" }}

```

### **Comment**

```bash
{% comment %}
your_text
{% endcomment %}

    Examples:
    --------
    {% comment %}
    My name is shama
    {% endcomment %}

```
